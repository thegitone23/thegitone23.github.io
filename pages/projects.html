Yet Another Kilo

A minimalistic terminal based text editor with zero external depedencies

This editor has been inspired by - the "kilo" text editor

Apart from the basic viewing and editing operations, kilo also supports basic syntax highlighting and searching.

So far, this "Yet Another Kilo" implementation extends a bit on syntax highlighting and improves searching.
